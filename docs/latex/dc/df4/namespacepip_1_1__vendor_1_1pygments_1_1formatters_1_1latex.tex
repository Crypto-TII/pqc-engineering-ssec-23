Small explanation of the mess below \+:) 

The previous version of the La\+TeX formatter just assigned a command to each token type defined in the current style. That obviously is problematic if the highlighted code is produced for a different style than the style commands themselves.

This version works much like the HTML formatter which assigns multiple CSS classes to each  tag, from the most specific to the least specific token type, thus falling back to the parent token type if one is not defined. Here, the classes are there too and use the same short forms given in \doxylink{namespacepip_1_1__vendor_1_1pygments_1_1token_af0f2c8141797724d5c7c715080bf2d52}{token.\+STANDARD\+\_\+\+TYPES}.

Highlighted code now only uses one custom command, which by default is \textbackslash{}\+PY and selectable by the commandprefix option (and in addition the escapes \textbackslash{}\+PYZat, \textbackslash{}\+PYZlb and \textbackslash{}\+PYZrb which haven\textquotesingle{}t been renamed for backwards compatibility purposes).

\textbackslash{}\+PY has two arguments\+: the classes, separated by +, and the text to render in that style. The classes are resolved into the respective style commands by magic, which serves to ignore unknown classes.

The magic macros are\+:
\begin{DoxyItemize}
\item \textbackslash{}\+PY@it, \textbackslash{}\+PY@bf, etc. are unconditionally wrapped around the text to render in \textbackslash{}\+PY@do. Their definition determines the style.
\item \textbackslash{}\+PY@reset resets \textbackslash{}\+PY@it etc. to do nothing.
\item \textbackslash{}\+PY@toks parses the list of classes, using magic inspired by the keyval package (but modified to use plusses instead of commas because fancyvrb redefines commas inside its environments).
\item \textbackslash{}\+PY@tok processes one class, calling the \textbackslash{}\+PY@tok@classname command if it exists.
\item \textbackslash{}\+PY@tok@classname sets the \textbackslash{}\+PY@it etc. to reflect the chosen style for its class.
\item \textbackslash{}\+PY resets the style, parses the classnames and then calls \textbackslash{}\+PY@do.
\end{DoxyItemize}
